## install

```Shell
npm install jsonldstore
```

## test

```Shell
npm test jsonldstore
```

## run

```Shell
usage: jsonldstore [options]

options:
  -p                 Port to use [8080]
  -a                 Address to use [0.0.0.0]
  -d                 JSON file with data to load
  -s --silent        Suppress log messages from output
  -h --help          Print this list and exit
```
